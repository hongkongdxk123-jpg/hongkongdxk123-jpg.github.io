📄前沿论文
arXiv Position: Hippocampal Explicit Memory Is the Cornerstone for AGI
观点论文：将外显记忆（类似人脑海马体的记忆机制）整合到 LLM 中是通向 AGI 的关键。认为当前模型缺乏的正是这种结构化、可检索的长期记忆能力。
arXiv Can AI Agents Synthesize Scientific Conclusions?
引入 SciConBench 基准测试，评估 AI Agent 在医疗等高风险领域综合科学证据并得出结论的能力。结果指出：AI Agent 在科学综合推理上仍有显著局限。
arXiv INFRAMIND: Infrastructure-Aware Multi-Agent Orchestration
提出基础设施感知的多 Agent 编排方法。现有方法只考虑任务和模型特征，忽略运行时基础设施状态。INFRAMIND 将服务器负载和网络延迟纳入 Agent 分配决策。
