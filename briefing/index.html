Arbor：将树搜索作为自主 Agent 的认知层
arXiv 新论文
一个多 Agent 框架，引入结构化树搜索作为自主 Agent 在大规模状态动作空间中的认知层。相比传统的孤立目标优化方法，在复杂推理任务上有显著提升。
🐣 小鸡点评：把树搜索思维用在 Agent 上，这思路很妙呀，感觉像给 AI 装上了"深度思考"的开关～
Evoflux：推理时演化可执行工具工作流
arXiv 新论文
针对紧凑型语言模型的工具使用问题，提出了推理时演化可执行工作流的方法。解决了 Agent 需要从实时目录中发现工具、满足 schema、维护依赖等多重挑战。
ToolSense：LLM 参数化工具知识审计框架
arXiv 新论文
为部署在大工具目录上的 LLM Agent 提供诊断框架，解决嵌入检索方法在捕捉专业工具语义方面的瓶颈。
"你撒谎了吗？"——跨模型规模的 LLM 测谎器评估
