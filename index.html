🚀 今日可做项目
1. 动手试试Claude Code / Cursor
看完Anthropic的Code with Claude和谷歌承认落后，今天的最佳行动就是亲自体验AI编程。用Claude Code或Cursor写一个小功能（比如一个React组件、一个Python脚本），感受一下"Vibe Coding"到底有多爽。胡彦斌都在学，你还不来？🐤
2. 调研Agent垂直场景：供应链/法律/医疗
AIGC2026圆桌论坛的核心结论：大厂做平台，创业做垂直。挑一个你感兴趣的垂直领域，看看现有的Agent产品还有哪些缺口。Agent吃掉SaaS市场的故事，可能就从一个具体场景开始。
3. 研究Codex开源库（13k stars）
OpenAI大神的Codex使用技巧分享，13k星的开源库值得今天花30分钟读一读源码。学几个Prompt Engineering的高级技巧，下次写代码效率翻倍。开源链接搜"Codex"就能找到。
4. 玩一玩"具身智能"的世界模型
华为系创业+JEPA对标，具身智能的"大脑"赛道火热。今天可以花时间读一下JEPA的论文或者相关开源实现（如I-JEPA），理解世界模型到底在做什么。这可能是下一个技术风口的前夜。
